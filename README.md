# DSA
Basic Concepts of DSA
This repository contains basic programs on Data Structures and Algorithms (DSA) implemented in C. It is designed to help beginners understand fundamental concepts through simple and practical examples.

📌 Topics Covered
🔢 Arrays
Traversal, insertion, deletion
Searching (Linear Search, Binary Search)
Sorting (Bubble, Selection, Insertion, Quick, Merge, Heap)
🔍 Searching Algorithms
Linear Search
Binary Search (on sorted arrays)
🔃 Sorting Algorithms
Bubble Sort
Selection Sort
Insertion Sort
Quick Sort
Merge Sort
Heap Sort
📚 Stack
Implementation using arrays
Push and Pop operations
Applications (expression evaluation, parentheses checking)
📥 Queue
Simple Queue
Circular Queue
Basic operations (enqueue, dequeue)
🔗 Linked List
Singly Linked List
Insertion (beginning, end, position)
Deletion operations
Traversal
🌳 Trees (Basic)
Binary Tree concepts
Tree traversal (Inorder, Preorder, Postorder)
📈 Algorithm Analysis
Time Complexity (Big-O notation)
Space Complexity basics
🚀 How to Run
Clone the repository:
git clone https://github.com/your-username/dsa-basic-programs.git
Navigate to the folder:
cd dsa-basic-programs
Compile the program:
gcc filename.c -o output
Run the program:
./output
